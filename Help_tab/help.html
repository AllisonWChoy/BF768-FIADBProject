Plots Page

This page will return plots such as bar charts, line charts, histograms, and bubble charts to explore interesting variables and trends in the Massachusetts FIA Tree database. The charts below are rendered by Google Charts API. Just click the dropdown menu to populate different plots to see relationships between different variables in the FIA database! For example, Tree Abundance and Species Abundance displays the number of Trees in each plot and the number of Species in general. Total Sum Biomass allows users to see the relationship between CarbonBG and CarbonAG in pounds for each year, the size of the bubble refers to tree population, so the bigger the bubble, the more trees there are in that year! This type of graph can also be seen in Total Sum Mortality vs Growth, instead its plotting MORTCFAL against GROWCFAL for each year. Average Growth and Average Mortality are shown in a grouped barchart and linechart.
